# CleanPool — Landing Page

Landing page de CleanPool. Frontend en **Vite + React + Tailwind CSS**.

## Ejecutar localmente

```bash
cd landingpage
npm install
npm run dev
```

Abre la URL que muestra Vite (normalmente `http://localhost:5173`).

## Build de producción

```bash
npm run build
npm run preview
```

## Contenido

- Hero con imagen del flotador (`imagen/flotador.png`)
- Grid de 6 funcionalidades con tarjetas 3D
- Carrusel del flujo medir → analizar → avisar → actuar
- Sección de beneficios
- Español / Inglés
